# gs-pensamento-computacional-e-automacao-com-python

## Integrante
 Rubens Henrique Cordeiro dos Santos - RM:572667
 
Esse codigo foi feito para simular um monitoramento de uma missão espacial.

O programa analisa diferentes ciclos da missão, 
verificando informações como temperatura, comunicação, bateria, oxigênio e estabilidade operacional.
Com base nesses dados, o sistema gera alertas, calcula níveis de risco, identifica tendências da missão e produz um relatório final para auxiliar 
na tomada de decisão.
